Amel Ship

Informasi Umum

Aplikasi Amel Ship dikembangkan oleh Yesica Tiara. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda.

Terakhir diperbarui: Maret 2026

Pengumpulan Data

Amel Ship tidak mengumpulkan, menyimpan, atau mentransmisikan data pribadi apapun.

Game ini beroperasi sepenuhnya secara offline dan tidak memerlukan:

  • Registrasi pengguna
  • Login akun
  • Koneksi internet (untuk gameplay)
  • Pelacakan aktivitas pengguna

Data Game Lokal

Data game (seperti skor tinggi, pembelian, dan pengaturan) disimpan secara lokal di perangkat Anda menggunakan teknologi penyimpanan lokal. Data ini tidak pernah dikirim ke server kami atau pihak ketiga manapun.

Layanan Pihak Ketiga

Aplikasi ini mungkin mengandung aset pihak ketiga (gambar dan audio) yang digunakan di bawah lisensi masing-masing. Kami tidak membagikan data pengguna apapun kepada pihak ketiga.

Anak-Anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami dengan sengaja tidak mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun.

Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Setiap perubahan akan diindikasikan dengan memperbarui tanggal "Terakhir diperbarui" di atas.

Hak Anda

Karena kami tidak mengumpulkan data pribadi, tidak ada data yang perlu diakses, diperbarui, atau dihapus. Namun, Anda dapat uninstall aplikasi ini kapan saja untuk menghapus semua data game lokal dari perangkat Anda.
